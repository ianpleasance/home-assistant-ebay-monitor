# eBay Integration for Home Assistant

Monitor your eBay account activity from Home Assistant. Track active bids, watchlist items, purchases, and create custom search alerts with automatic new-item detection.

## Version 2.3.0

### Features

- **Active Bids** — Real-time monitoring with high bidder status, reserve met, and time remaining
- **Watchlist** — Track watched items with price and watcher count
- **Purchases** — Purchase history with shipping status and tracking numbers
- **Saved Searches** — Create multiple searches with configurable intervals and automatic new-item alerts
- **Smart Won/Lost Detection** — Auction results verified via eBay API with grace period to prevent false alerts
- **Multi-Account** — Multiple eBay accounts supported independently
- **API Usage Tracking** — Monitor eBay API usage in real time

## Languages

Danish, Dutch, English, Finnish, French, German, Italian, Japanese, Norwegian, Polish, Portuguese, Spanish, Swedish

## Requirements

- Home Assistant 2024.1.0 or newer
- eBay Developer account with API credentials (free)
